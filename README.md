# Chitai-Gorod 

Этот проект содержит автоматизированные тесты для API Читай-город. Тесты проверяют функциональность поиска товаров через API.

## Установленные зависимости:
- allure-pytest
- pytest
- selenium
- requests
- webdriver-manager
- python-dotenv
